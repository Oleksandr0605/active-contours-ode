**Тема**: Моделювання активних контурів (Snakes) як динамічної системи: від PDE до програмної реалізації.  
**Мета проєкту** — реалізувати еволюцію замкненого контуру, який поводиться як пружна нитка у в’язкому середовищі під дією зовнішніх сил (градієнтів зображення).  
**Математична суть**: В основі лежить рівняння в частинних похідних 4-го порядку, що мінімізує енергію системи:$$\frac{\partial \mathbf{x}}{\partial t} = \alpha \frac{\partial^2 \mathbf{x}}{\partial s^2} - \beta \frac{\partial^4 \mathbf{x}}{\partial s^4} - \nabla P(\mathbf{x})$$
Це, рівняння дифузії з додаванням зовнішнього силового поля.  

**План**: Перетворити це PDE на систему зв'язаних звичайних диференціальних рівнянь (ОДУ) для $N$ вузлів контуру:$$\dot{\mathbf{V}} = -A\mathbf{V} + \mathbf{F}_{ext}$$Де матриця $A$ — це дискретний оператор кривизни (внутрішні сили).  

**Розв'язок**: Реалізувати програму на основі напівнеявної схеми (щоб система була стійкою і не "вибухала" при великих кроках часу).

**Аналіз динамічної системи**:
 - Як власні числа матриці $A$ впливають на швидкість збіжності та згладжування шуму.
 - Залежність стійкості від параметрів еластичності ($\alpha$) та жорсткості ($\beta$).
